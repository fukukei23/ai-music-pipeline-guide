04 video-prompt-spec — 映像プロンプト仕様書

入力（テキストアイデア/音源/動画/YouTube URL）を映像制作用の4構造プロンプト仕様書に変換。楽曲のMV・プロモーション映像・ショート動画の映像化に。

できること

入力から映像制作AIが使える4構造のプロンプト仕様書を出力:

  1. 映像タイムライン — シーン構成・時間軸
  2. ビジュアル特徴・画風 — 色調・構図・ムード
  3. 画像生成AI用プロンプト（英+訳）— ジャケット・キービジュアル用
  4. 動画AI用モーション指示 — カメラワーク・動き

トリガー

「映像プロンプト」「動画プロンプト」「映像仕様書」「ビジュアルプロンプト作って」「MVのプロンプト」「/video-prompt-spec」

設計の核心: 映像に特化

💡 本スキルは**「映像」専用**。音楽生成プロンプト（Suno/Udio用のStyle/Tags/Lyrics）は出力しない。それが必要なら reverse-engineer-song へ。

役割分離を明確にすることで、各AIの得意領域を活かす。

入力と処理

入力 処理
テキストアイデア CC 単独で処理
音源・動画・YouTube URL Gemini（gemini-code）に分析委任

CC は音源/動画を直接扱えないため、Gemini API 経由で分析を委任する点は reverse-engineer-song と同じ設計。

連携

  • 上流: make-song で作った曲（音源）を入力にMV映像を企画
  • 並列: reverse-engineer-song の音楽プロンプト + 本スキルの映像プロンプトで楽曲+映像を完全カバー
  • 出力先: 画像生成AI（Midjourney/SD等）・動画AI（Runway/Pika等）
